# Product Overview

This is a **Social Media Scraper** focused on phishing detection and security analysis. The system extracts URLs from Reddit, Instagram, and X (Twitter) to identify potentially malicious links and suspicious domains.

## Core Purpose
- Extract URLs from social media posts across multiple platforms
- Identify shortened URLs and suspicious domains for phishing detection
- Provide unified interface for cross-platform security analysis
- Support persistent login sessions and browser automation

## Key Features
- Multi-platform URL extraction (Reddit, Instagram, X)
- Phishing detection with risk scoring
- Session management and authentication
- Async operations for high-performance scraping
- Integration with GUI application for automated scanning

## Target Use Cases
- Security research and threat detection
- Phishing account identification
- Malicious link analysis across social platforms
- Educational security analysis tools
